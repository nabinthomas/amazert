

1. Firebase console - Add project [ Follow Firebase setup: https://firebase.google.com/docs/android/setup ]
2. Android — Register App - Download ‘google-services.json’
3. Copy the google-services.json to app/ folder 
4. Project/App level Gradle file changes: 
	https://console.firebase.google.com/u/0/project/amaze-8f94e/overview
5. Ensure step4 “Run your app to verify installation” completes successfully and 
successfully adds Firebase to App :: “Congratulations, you've successfully added Firebase to your app!”
6. SHA1 
	https://developers.google.com/android/guides/client-auth
Firebase Console — Settings — Project settings - Add fingerprint 
	
7. Enable Google Sign-In in the Firebase console:
	a.	In the Firebase console, open the Auth section.
	b.	On the Sign in method tab, enable the Google sign-in method and click Save.
8. Create RealTime Database 
	Firebase Console —> Realtime Database —> Create Database —> 
	Select “Start in Test Mode” - i.e. with read/write access for the show world
